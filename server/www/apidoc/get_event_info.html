    Description
    The get_event_info API returns basic data about a specific event in text format.
    URL
    http://trideccloud.gfz-potsdam.de/api/get_event_info
    Input parameter
    Property Description Status
    apikey A string of 32 characters that authorizes the usage of this API. It is linked to a user or institution account. Mandatory
    evid The unique ID of the event for which information should be retrieved. The latest refinement will be returned, if this is a GEOFON identifier. Mandatory
    Return value
    Property Description Status
    status States the status of the request. Can be one of the following:
    • success: The operation was successfully accomplished. The values of the remaining fields are valid.
    • failure: An error occurred while trying to apply the request. This may be caused by invalid input parameter values.
    • denied: The access to this API was denied. The given API key is incorrect or not valid in conjunction with the API used.
    		Always valid.
    evid Contains the unique event ID used by the TRIDEC Cloud platform. Valid if status is set to success.
    geofonid Contains the official event ID specified by the GEOFON Program. Valid if status is set to success.
    prop Lists all properties available for this earthquake event. Valid if status is set to success.
    simulation Includes information about a processed tsunami simulation based on the given event parameters. Only present if a computation was requested. Valid if status is set to success.
    msg Contains an unapproved exercise message that has been automatically generated. Valid if status is set to success.
    prop
    Property Description Status
    region The region in which the earthquake occurred. Valid if status is set to success.
    date The origin time of the earthquake in UTC. The date is given in ISO format: yyyy-MM-dd'T'HH:mm:ss.SSS'Z'. Valid if status is set to success.
    magnitude The magnitude of the earthquake. Valid if status is set to success.
    latitude The latitude of the earthquake in the range of -90 to +90 degrees. Valid if status is set to success.
    longitude The longitude of the earthquake in the range of -180 to +180 degrees. Valid if status is set to success.
    depth The depth of the earthquake in kilometer. Valid if status is set to success.
    dip Fault dip of the earthquake in the range of 0 to 90 degrees. Can be null if moment tensor is not available. Valid if status is set to success.
    strike Fault strike of the earthquake in the range of 0 to 360 degrees. Can be null if moment tensor is not available. Valid if status is set to success.
    rake Rake of the earthquake in the range of -180 to +180 degrees. Can be null if moment tensor is not available. Valid if status is set to success.
    sea_area Contains a label of the sea region in which the earthquake occurred. Can be null if the event is situated on land. Valid if status is set to success.
    simulation
    Property Description Status
    resolution Resolution of the underlying grid in arc minutes. Valid if status is set to success.
    resources Lists the computational resources used to perform the simulation. Valid if status is set to success.
    grid_dim Includes the coordinates of the bounding box used for the simulation. It contains the sub-fields latMin, latMax, lonMin and lonMax that define the boundaries in degree. Valid if status is set to success.
    simTime Number of minutes for which the tsunami propagation has been simulated. Valid if status is set to success.
    calcTime States the amount of time (in milliseconds) needed to complete the simulation. Valid if status is set to success.
    progress Progress of the computation. Set to 100 if the simulation has been completed already. Valid if status is set to success.
    curSimTime States for how many minutes the simulation has been performed at the moment. Valid if status is set to success.
    shared_link Contains a dynamic url to the TRIDEC Cloud platform that can be used to view the simulation results and accomplish further analyses. Valid if status is set to success.
    image_url Specifies a dynamic url that points to an image of the simulation result. Valid if status is set to success.
