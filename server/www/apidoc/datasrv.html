    Description
    The datasrv API serves data for a specific event. Data can be anything from raw data to final end user products associated with an event.
    URL
    http://trideccloud.gfz-potsdam.de/datasrv/<EVENT-ID>/<PRODUCT-TYPE>?apikey=<API-KEY>&download=<FILE-NAME>
    URL Path Elements
    Property Description Status
    <EVENT-ID> The unique ID of the event for which information should be retrieved. The latest refinement will be returned, if this is a GEOFON identifier. Mandatory
    <PRODUCT-TYPE> The name of the specific product. Following products can be retrieved in different resolution and format:
    • A map with Wave Jets and Travel Times in web quality (web) or high quality (hq) as PNG or PDF file, e.g. wavejets_traveltimes_hq.pdf generates a PDF file with Wave Jets and Travel Times in 600dpi resolution.
    • A map with Coastal Forecast Zones (CFZs) and Tsunami Forecast Points (TFPs) in web quality (web) or high quality (hq) as PNG or PDF file, e.g. cfzs_tfps_web.png generates a PNG file with CFZs and TFPs in 300dpi resolution.
    • A map with Cities and Population Density information in web quality (web) or high quality (hq) as PNG or PDF file, e.g. cities_population_hq.png generates a PNG file in 600dpi resolution.
    • Basic earthquake paramters as CSV file, i.e. eq.csv.
    • Text information on the computed maximum wave height and the earliest arrival for Tsunami Forecast Points (TFPs) as CSV file, i.e. tfps.csv.
    • Text information on the computed maximum wave height and the earliest arrival for Coastal Forecast Zones (CFZs) as GMT file, i.e. cfzs.gmt.
    Other products such as custom maps or wave forms are available, too. To receive a complete list of available products with short descriptions please open http://trideccloud.gfz-potsdam.de/datasrv/ in your browser.
    To receive an incomplete access list of products for a specific event please use http://trideccloud.gfz-potsdam.de/datasrv/<EVENT-ID>     		Mandatory
    URL Query Parameters
    Property Description Status
    apikey A string of 32 characters that authorizes the usage of this API. It is linked to a user or institution account. Note: In case the API is used in the browser and you are already signed-in then the API key is not required. Mandatory
    download The file name for the product to be downloaded. If the download parameter is given but empty, then the filename will be automatically generated from the <EVENT-ID> with the proper file extension. Note: In case the API is used in the browser and you omit this parameter, then the product might be displayed in the browser (in case of CSV, PDF, and PNG products ). Optional
    Return Value
    The requested product or data will be returned as a file. In case of any malfunction the respective HTTP error response with its HTTP status code will be returned, e.g. 404 for an unknown <EVENT-ID> or an unknown <PRODUCT-TYPE>.
    API Examples
    Near East Coast of Honshu, Japan - March 11, 2011
    (please sign-in first, or use your API key):

    http://trideccloud.gfz-potsdam.de/datasrv/gfz2011ewla/eq.csv?apikey=***

    http://trideccloud.gfz-potsdam.de/datasrv/gfz2011ewla/wavejets_traveltimes_web.png?apikey=***
