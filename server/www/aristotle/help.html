Search Help

By default, the search performs a logical OR operation based on the given terms. There exist some advanced search operators which can be used to guide the search according to your needs. The following table gives an overview of the available operators.
Operator Definition Example
"" (quotes) Performs an exact search and can be used to combine terms with AND semantics. Example 1: "dust storm"
Meaning: Search for items that include the exact phrase "dust storm".

Example 2: "dust" "sand"
Meaning: Search for items that include both terms "dust" and "sand".
in:institute
in:office
in:person
in:decision
in:advice		 Search for items in institutes, offices, persons, decisions or advices. Example: in:office earthquake storm
Meaning: Search for offices which are responsible for earthquakes or storms.
display:institute
display:office
display:person
display:decision
display:advice		 Limit results to institutes, offices, persons, decisions or advices. Example: display:office in:person "crisis coordinator"
Meaning: Show all offices which have a crisis coordinator assigned.
id: List all items that belong to the institute identified by the acronym provided. Example: id:ERCC
Meaning: Show all items that are related to the Emergency Response Coordination Centre (ERCC).
